## 🧮 Calculator Features

- Basic arithmetic operations (+, −, ×, ÷)
- Supports BODMAS rule
- Scientific functions: `sin`, `cos`, `tan`, `sqrt`, `log`, `π`, `^`
- Ratio and percentage calculations
- Keyboard input support
- History tracking of previous calculations
